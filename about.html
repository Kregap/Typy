About Typy

Typy is allows to take short notes on your desktop with Markdown.

How it works:
  • Start typing in the editor (edit mode is always default).
  • Switch to preview mode to see your formatted Markdown.
  • Click anywhere in preview to return to editing.
Available Shortcuts:
  • Esc or Right Click: Switch to preview mode
  • Click (in preview): Return to edit mode
  • Ctrl + Right Click: Move window
  • Ctrl + Mouse Wheel: Zoom in/out
  • Shift + / (?): Show this help/about popup
  • Ctrl + Left Click: Close app
